# MicroVM / Firecracker 研究筆記

---

## 🔥 最新動態 2026-06-27

- AWS Lambda MicroVMs 正式發布 (AWS News Blog)
  摘要：AWS 推出 Lambda MicroVMs，一個全新的 serverless compute primitive，讓開發者可以在隔離的、有狀態的執行環境中運行用戶或 AI 生成的代碼。基於 Firecracker 技術，提供 VM 級別隔離、近乎即時的啟動和恢復、完整的生命週期控制，狀態可保留長達 8 小時。無需管理基礎設施或複雜虛擬化技術。Firecracker 已經支撐了超過 15 萬億次每月 Lambda 函數調用。
  價值：這是 Firecracker 技術的重大產品化里程碑，將 MicroVM 隔離能力從 AWS 內部基礎設施擴展到用戶可編程的 serverless 沙箱。對於需要運行不可信用戶代碼的場景（AI coding assistants、互動式代碼環境、數據分析平台、漏洞掃描器、遊戲服務器運行用戶腳本）提供了完美的解決方案。

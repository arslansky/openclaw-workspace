訪問總結｜蔡金強（小金人）
etnet 影片訪問｜2026-07-07｜21 分 42 秒
========================================================

【一句話總結】
今年（2026）股市賺唔到錢，唔係你揀錯 — 係嗰個市場本身「唔當做」。
當做嘅市場，全部都係一個主題：AI 硬件（特別係半導體）。

【三大重點】

1) 「今年要睇個市場當唔當做」
   - 唔係港股得唔得、印度得唔得嘅問題
   - 係嗰個市場入面有冇 AI、冇 AI 硬件、冇半導體
   - 當做嘅市場：美國、韓國、台灣、日本、A 股（仲可以）
   - 唔當做：印度、歐洲、英國、東南亞
   - 港股：AI 硬件佔比只係 10% 出頭，跟印度、歐洲差唔多 → 「冇得攪」

2) 真正嘅「AI hardware Big boss」只係幾隻
   - GPU → NVIDIA
   - DRAM / HBM → SK Hynix、Samsung
   - 代工 → TSMC
   - 呢幾隻先係 supply chain 嘅主角
   - 蘋果嘅位置「被大大削弱」— 因為 AI 電話用到嘅 memory 成本會急升

3) 韓國 memory（SK Hynix / Samsung）嘅角色
   - 韓國靠 memory 兩隻股撐起成個市場
   - 即使出咗新嘢（敘述被截斷），GPU 主宰嘅世代唔會變
   - 韓國 memory 嘅位置 ≈ 將來嘅 NVIDIA

【投資啟示】

- 今年揀市場，先睇「當唔當做 AI 硬件主題」
- 港股：AI 硬件佔比低，唔使強求
- 真正主角集中：NVIDIA、SK Hynix、Samsung、TSMC
- 防守股、街坊思維：內心要有準備個市「可以係唔郁」
- 情緒化判斷「港股得 / 唔得」冇意義 — 睇結構先

【一句到尾】
今年係 AI 硬件半導體嘅世界，其它市場陪坐。港股？執啲穩嘅冇問題，執中位就小心。
